# npoqui

> Made with create-react-library

## Install

```bash
npm install --save npoqui
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'npoqui'
import 'npoqui/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
